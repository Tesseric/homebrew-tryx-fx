# homebrew-tryx-fx

Homebrew tap for [TryX FX](https://github.com/Tesseric/tryx-fx) audio plugins.

## Install

```bash
brew tap Tesseric/tryx-fx
brew install --cask tryx-fx
```

## What's included

- **Tryxs Time Fracture** — Creative delay effect with randomized delay, filtering, distortion, and pitch shifting (VST3, AU, Standalone)
- **TryX Synth** — AI-powered multi-engine synthesizer with Gemini LLM integration (VST3, AU, Standalone)
